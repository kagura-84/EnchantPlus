# EnchantPlus
EnchantPlus is a Minecraft data pack that adds many original enchantments.

## Dependency
This datapack requires a [CustomEnchant-Core](https://github.com/kagura-84/CustomEnchant-Core).

### Version
正常な動作確認済み: 1.19.2

動作確認中: 1.19.1, 1.19.3
